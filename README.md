## 🛠 Установка (в одну команду)

Чтобы установить SolverAI в систему и запускать её по одному слову, открой **PowerShell от имени Администратора** и вставь следующую команду:

```powershell
iwr -Uri "ТВОЯ_ПРЯМАЯ_ССЫЛКА_НА_EXE" -OutFile "$env:SystemRoot\SolverAI.exe"
```

## 🚀 Как пользоваться
После установки тебе не нужно искать файлы. Просто нажми `Win + R` или открой любую консоль и напиши:
**SolverAI**
